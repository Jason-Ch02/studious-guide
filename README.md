- **IDE**: Visual Studio 2022&2026
- **语言标准**: C++ 11 或更高版本
- **系统**: Windows 11

## 学习笔记
学习了如何将 `.h` (头文件) 与 `.cpp` (源文件) 分离，让代码更易维护。
掌握了如何过滤临时文件，保持仓库纯净。
对类和对象有一定的理解，能够理解且运用对象的封装，继承和多态。
对理解并能运用写入文件和读取文件等关于文件的代码
