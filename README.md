## 📂 当前精品项目

### 1️⃣ 圆与点的关系 (Point-Circle Relationship)
> **技术栈**：类与对象、封装、构造函数、分文件编写

- **项目描述**：通过输入圆心坐标、半径以及点坐标，计算并判断点与圆的位置关系。
- **数学逻辑**：
  - 计算点到圆心的距离 $d$。
  - $d < r$ ：点在圆内。
  - $d = r$ ：点在圆上。
  - $d > r$ ：点在圆外。
- **文件结构**：
  - `点类.h / .cpp`：定义坐标属性。
  - `圆类.h / .cpp`：定义半径及判断逻辑。

### 2️⃣ 通讯录管理系统 (Address Book)
> **技术栈**：结构体数组、指针、循环分支逻辑

- **项目描述**：实现了一个基础的后台管理逻辑，包含：
  - [x] 添加/显示联系人
  - [x] 删除/修改联系人
  - [x] 查找/清空联系人
